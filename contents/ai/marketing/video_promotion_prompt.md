# 短视频生成提示词：人机协作的四个进化阶段

## 1. 核心信息提取 (Core Analysis)
- **主题**：AI 编程/协作的进化论（从“司机”到“指挥官”）。
- **隐喻**：驾驶汽车（Driving Metaphor）。
- **四个阶段**：
  1.  **独行 (Human Driving)**：累、全责。
  2.  **辅助 (Copilot)**：副驾提醒，主驾决策。
  3.  **共驾 (AI Pilot)**：AI 开车，人指挥/兜底。
  4.  **自动 (Auto Pilot)**：人定目标，AI 全包。
- **结论**：关键在于信任与沟通（Prompting），从执行者转变为决策者。

## 2. 视频生成提示词 (Video Generation Prompt)

### 📋 视频元数据
- **时长**：45-60秒
- **平台**：TikTok / Shorts / 视频号
- **风格**：快节奏、科技感、分屏对比、视觉隐喻强
- **BGM**：
  - 前10秒：快节奏、略带紧张感的机械打字声/时钟声。
  - 中段：科技感电子乐，逐渐昂扬。
  - 结尾：宏大、空灵的未来感合成器音乐。

## 3. 制作建议 (Production Notes)
- **剪辑节奏**：卡点剪辑，每个阶段切换时要有明显的音效（如“Whoosh”）。
- **数字人/旁白**：建议使用稍微轻快、有科技感的年轻男声或女声。
- **封面图**：使用阶段一（累）和阶段四（爽）的左右分屏对比图，标题写“AI 编程：你是司机还是指挥官？”。

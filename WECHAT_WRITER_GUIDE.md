## 快速开始

### 1. Agent 配置文件已创建 ✅

**文件位置**：`.claude/agents/wechat-writer.md`

- 工作流程和输出规范
- 质量检查清单

---

### 2. 如何使用 WeChat Writer Agent

#### 方法 A：在对话中直接调用

```
请使用 wechat-writer agent 创作一篇关于港股 IPO 打新策略的文章，要求：
- 文章类型：投资指南
- 目标受众：普通投资者
- 预计字数：2000 字
- 系列位置：独立文章（非系列）
```

#### 方法 B：使用 Task 工具

```bash
task(
  category="writing",
  load_skills=[],
  prompt="创作一篇关于港股 IPO 打新策略的公众号文章，文章类型：投资指南，目标受众：普通投资者，预计字数：2000 字"
)
```

---

### 3. 输入信息清单

在调用 wechat-writer agent 时，请提供以下信息：

| 必需 | 信息项 | 说明 | 示例 |
|------|--------|------|------|
| ✅ | 主题 | 文章的核心主题 | 港股 IPO 打新策略 |
| ✅ | 目标受众 | 文章的目标读者 | 普通投资者 |
| ✅ | 文章类型 | 文章的类型 | 科普/分析/案例/教程/指南 |
| ❌ | 系列位置 | 文章在系列中的位置 | 独立/Part 1/Part 2/前传 |
| ❌ | 预计字数 | 预期的字数范围 | 2000 字 |

**示例输入**：
```
主题：港股 IPO 打新策略
目标受众：普通投资者
文章类型：投资指南
系列位置：独立文章
预计字数：2000 字
```

---

### 4. 输出内容

WeChat Writer Agent 会生成一篇完整的公众号文章，包含：

#### 标题
- 双标题结构（主标题 + 副标题）
- Emoji 点缀
- 系列编号（如适用）

#### 开头
- Blockquote 引入
- 系列导航（如适用）
- 摘要/导读
- 设问（可选）

#### 正文
- 3-5 个章节
- 每章节有 emoji 标记
- 案例驱动
- 数据支持
- 生动比喻

#### 视觉元素
- 2-3 个图表
- 关键信息加粗
- 引用块
- 分隔线
- 列表
- 数学公式（如适用）

#### 结尾
- 总结
- 价值升华
- 系列预告（如适用）
- 互动引导

#### 互动元素
- 话题标签（3-5 个）
- 免责声明
- 作者信息

---

### 5. 验证和优化

生成文章后，建议进行以下验证：

#### 第一步：对照写作风格指南

打开 `docs/writing-style-guide.md`，对照质量检查清单逐项检查。

#### 第二步：与现有文章对比

选择一篇参考文章，对比风格、结构、内容是否一致。

#### 第三步：使用测试用例验证

打开 `docs/wechat-writer-test-cases.md`，选择合适的测试用例进行验证。

#### 第四步：人工审查

人工阅读文章，检查可读性、准确性、价值性等。

---

### 6. 优化和调整

如果验证发现问题，可以：

1. **调整输入信息**：提供更详细的要求
2. **参考模板文件**：使用 `templates/article-template.md` 作为参考
3. **查阅风格指南**：查看 `docs/writing-style-guide.md` 中的详细规范
4. **重新生成**：调整后重新调用 agent 生成

## 常见问题

### Q1：如何确保生成的文章符合风格？

**A**：
1. 提供详细的输入信息（主题、受众、类型等）
2. 参考写作风格指南 (`docs/writing-style-guide.md`)
3. 参考现有文章风格
4. 使用文章模板 (`templates/article-template.md`)

### Q2：生成的文章质量不理想怎么办？

**A**：
1. 调整输入信息，提供更详细的要求
2. 参考测试用例，对照预期输出
3. 重新生成，强调不符合要求的部分
4. 手动修改和优化

### Q3：如何验证生成的文章？

**A**：
1. 使用 4 种验证方法（详见 `docs/wechat-writer-verification.md`）
2. 对照写作风格指南逐项检查
3. 与现有文章对比风格
4. 使用测试用例验证

### Q4：可以自定义写作风格吗？

**A**：可以。修改以下文件：
- `.claude/agents/wechat-writer.md` - 修改系统提示词
- `docs/writing-style-guide.md` - 修改风格指南
- `templates/article-template.md` - 修改模板

### Q5：如何生成系列文章？

**A**：在输入信息中指定系列位置：
- 第一篇：系列位置 = "前传" 或 "Part 1"
- 中间篇：系列位置 = "Part 2", "Part 3", ...
- 总结篇：系列位置 = "Part 5" 或 "总结"

Agent 会自动：
- 生成系列导航
- 引用前文链接
- 预告后续内容

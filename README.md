# Writing Projects

This repository contains various writing projects and articles.

## Directory Structure

- **`contents/`**: Main content directory.
  - **`finance/`**: Finance and IPO related content.
    - **`greenshoe/`**: Content related to the "Hong Kong IPO Greenshoe Mechanism" (港股IPO绿鞋机制).
      - `content/`: Markdown source files.
      - `assets/`: Images and other static assets.
      - `scripts/`: Python scripts for generating charts or maintenance.
    - **`ipo/`**: General IPO articles.
      - `content/`: Markdown source files (e.g., `ipo_allocation.md`).
      - `assets/`: Related assets.
  - **`ai/`**: AI Technology Series.
    - `content/`: AI related articles.
    - `assets/`: AI related assets.

- **`archives/`**: Backups and archived content.
  - `wechat_backup/`: Backups of published WeChat Official Account articles.

## Scripts

Scripts are located within their respective project folders (e.g., `contents/finance/greenshoe/scripts/`).

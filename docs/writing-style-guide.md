本文档详细说明了公众号文章的写作风格规范，适用于金融投资科普、技术分析、案例复盘、AI 技术科普等内容创作。

## 主题支持

本项目支持两种主题风格，创作前请先识别主题类型：

| 主题类型 | 适用场景 | 风格特点 | 参考文章 |
|---------|----------|----------|----------|
| **金融主题** | 港股 IPO、投资理财、金融科普、案例复盘 | 双标题结构、案例驱动、数据支持、风险提示 | `contents/finance/greenshoe/content/` |
| **AI 主题** | AI 技术科普、人机协作、工具使用、技术趋势 | 单标题结构、比喻驱动、互动话题、配图建议 | `contents/ai/content/ai_collaboration_stages.md` |

### 金融主题风格快速参考

- **标题**：双标题（主标题 + 副标题）+ emoji
- **结构**：开头（blockquote + 导航 + 摘要）→ 章节编号 → 结尾（总结 + 升华 + 互动）
- **内容**：案例驱动 + 数据支持 + 生动比喻
- **互动**：话题标签 + 免责声明 + 点赞引导

### AI 主题风格快速参考

- **标题**：单标题（引导词 + 核心主题）+ emoji
- **结构**：【导语】→ 阶段划分（4个阶段）→ 【结语】→ 【互动话题】
- **内容**：比喻驱动 + 列表说明 + 配图建议
- **互动**：互动话题 + 投票选项 + 评论区邀请

## 主题风格对比

### 金融主题 vs AI 主题

| 维度 | 金融主题 | AI 主题 |
|------|----------|---------|
| **标题格式** | 双标题（主标题 + 副标题）+ emoji | 单标题（引导词 + 核心主题）+ emoji |
| **标题引导词** | 无 | "一文看懂"、"深度解析"、"全面了解" |
| **开头引入** | Blockquote + 系列导航 + 摘要 + 设问 | Blockquote（💡）+ 【导语】+ 比喻 |
| **章节划分** | 01/02/03 或 第一/二/三部分 | 阶段编号（第一阶段、第二阶段...）|
| **章节标记** | 数字编号 + emoji | Emoji + 阶段编号 + 阶段名称 |
| **特殊章节** | 使用 `---` 分隔 | 使用 【】标记（【导语】、【结语】、【互动话题】）|
| **内容驱动** | 案例驱动 + 数据支持 | 比喻驱动 + 列表说明 |
| **比喻使用** | 2-3 个生动比喻 | 贯穿全文的比喻（如"驾驶之旅"）|
| **视觉元素** | 图表 + 表格 + 公式 | 大量 emoji + 列表 + 配图建议 |
| **图表类型** | GitHub raw、Mermaid、在线图表 | 配图建议（描述性，非固定链接）|
| **互动元素** | 话题标签 + 免责声明 + 点赞引导 | 【互动话题】+ 投票选项 + 评论区邀请 |
| **结尾设计** | 总结 + 升华 + 系列预告 + 互动 | 【结语】+ 范式转移 + 展望未来 |
| **字数要求** | 1500-2500 字 | 1200-2000 字 |

- 文章主题与金融、投资、IPO 相关
- 需要使用真实案例和数据支持
- 需要专业的风险提示和免责声明

## 标题风格

### 双标题结构

**格式**:
```
# [主标题]：[副标题]
```

**要求**:
- **主标题**: 10-20字，吸睛、概括性强
- **副标题**: 20-30字，补充说明、吸引点击
- 使用冒号 `：` 分隔（中文全角冒号）

**示例**:
```
# 揭秘港股 IPO 的"护盘神器"：绿鞋机制究竟是救命稻草还是昙花一现？
```
```
# IPO真相：为什么有的新股"跌不动"？揭秘绿鞋与股权金字塔的博弈游戏
```

### 系列编号

**格式**:
```
# [编号] [标题]
```

**编号规则**:
- **前传/引子**: `00 前传：标题`
- **正式文章**: `01/02/03...` 或 `Part 1/Part 2/Part 3...`
- **复盘总结**: `05/06...` (根据实际位置编号)

**示例**:
```
# 00 前传：深入理解港股 IPO 中的绿鞋机制
```
```
# 01 揭秘港股 IPO 绿鞋机制实例解析 (一)
```
```
# Part 2：那些年我们追过的"绿鞋"
```
```
# 05 绿鞋机制深度复盘：29 只港股 IPO 实战全景与投资生存指南
```

### 吸引关键词

**列表**（每篇标题使用 2-3 个）:

| 类型 | 关键词 |
|------|--------|
| **揭示类** | 揭秘、深度解析、真相、深度、全景 |
| **实战类** | 实战全景、博弈游戏、实例解析、复盘 |
| **指南类** | 指南、生存指南、操作手册、实战手册 |
| **幕后台** | 幕后逻辑、幕后真相、隐藏规则 |

**组合示例**:
- "深度解析" + "幕后逻辑"
- "实战全景" + "博弈游戏"
- "揭秘" + "真相"

### Emoji 点缀

**Emoji 列表**（每篇标题使用 1-2 个）:

| Emoji | 含义 | 适用场景 |
|-------|------|----------|
| 🧐 | 探索、分析、理解 | 深度分析、理论解析 |
| 📉 | 趋势、数据、下跌 | 市场分析、趋势研判 |
| 💡 | 启示、建议、技巧 | 投资建议、技巧分享 |
| 🚀 | 成功、增长、突破 | 成功案例、增长分析 |
| 🛡️ | 保护、防御、风险 | 风险提示、防御策略 |
| 🧊 | 结构、基础、机制 | 机制解析、结构说明 |
| ⚔️ | 博弈、竞争、对抗 | 市场博弈、竞争分析 |
| 📝 | 总结、记录、文档 | 案例复盘、总结文章 |

**使用规则**:
- Emoji 放在标题末尾或关键位置
- 不要过度使用（1-2个即可）
- Emoji 要与标题内容相关

**示例**:
```
# 05 绿鞋机制深度复盘：29 只港股 IPO 实战全景与投资生存指南 📊
```
```
# 深度 | 揭秘港股 IPO "绿鞋机制": 稳价神器的幕后逻辑与实战博弈 🧐
```

**格式**:
```

```

**示例**:

**格式**:

**示例**:
```
## 01 什么是"绿鞋"?

"绿鞋"这个名字听起来很时尚，但它其实是**"超额配售选择权" (Over-allotment Option) **的俗称。这个机制本质上是承销商手中的一项选择权。

### 绿鞋的运作原理

简单来说，绿鞋机制就是为了稳定上市初期的股价而设计的一个"缓冲垫"。

它的运作原理大概是这样的：

1. **先借再卖**：上市时，承销商会多问大股东借 15% 的股票，按发行价卖给市场。
2. **看跌买回**：如果上市后股价跌了，承销商就用这笔钱从二级市场上把那 15% 的股票买回来。
3. **看涨行权**：如果上市后股价涨了，承销商就不需要从市场买股票了，而是直接行使"超额配售权"。

**案例展开**：

2025 年赛力斯港股上市，盘中一度跌超 9%。就在收盘前的关键几秒，稳价人投入数亿资金"暴力拉升"，将股价瞬间打回发行价平盘报收。这就是绿鞋资金在关键时刻的"定海神针"作用。
```

## 内容特点

### 语言风格

#### 口语化表达

**要求**:
- 使用"你看"、"试想一下"、"简单来说"等口语化表达
- 避免过于学术化的词汇
- 必要时给出通俗解释

**示例**:
```
❌ 错误：绿鞋机制是指在股票首次公开发行时，承销商有权超额配售一定比例的股票。

✅ 正确：简单来说，绿鞋机制就是为了稳定上市初期的股价而设计的一个"缓冲垫"。
```

```
❌ 错误：当发行价高于市场价时，承销商将不会行使绿鞋期权。

✅ 正确：简单来说，就是如果股价涨了，承销人就不买了，直接要求公司增发新股。
```

#### 生动形象的比喻

**要求**:
- 每篇文章至少 2-3 个生动比喻
- 比喻要与内容相关且贴切
- 使用加粗突出比喻

**常用比喻列表**:

| 比喻 | 含义 | 适用场景 |
|------|------|----------|
| 护盘神器 | 稳定股价的工具 | 绿鞋机制 |
| 遮羞布 | 掩盖问题的工具 | 无效的护盘 |

## 常见误区

### ❌ 误区 1：标题过于学术化

**错误示例**:
```
# 绿鞋机制对港股 IPO 价格稳定作用分析
```

**正确示例**:
```
# 揭秘港股 IPO 的"护盘神器"：绿鞋机制究竟是救命稻草还是昙花一现？
```

### ❌ 误区 2：章节编号混乱

**错误示例**:
```
## 1. 什么是绿鞋？
## 第二部分：绿鞋运作原理
## Part 3：实战案例
```

**正确示例**:
```
## 01 什么是"绿鞋"?
## 02 绿鞋是如何"稳价"的?
## 03 实战案例复盘
```

### ❌ 误区 3：过度使用 emoji

**错误示例**:
```
## 🧐📉💡🚀🛡️ 01 什么是绿鞋？
```

**正确示例**:
```
## 🧐 01 什么是"绿鞋"?
```

### ❌ 误区 4：没有数据支持

**错误示例**:
```
古茗的绿鞋消耗得很快，几天就没了。
```

**正确示例**:
```
古茗的绿鞋可谓是"鞠躬尽瘁"。数据显示，上市**第一天**，绿鞋资金就消耗了 **81.3%**！剩下的子弹在**四天内**全部打光。

数据来源：LiveReport 及市场公开数据
```

### ❌ 误区 5：比喻不贴切

**错误示例**:
```
绿鞋机制就像是一个巨大的盾牌，可以抵挡所有攻击。
```

**正确示例**:
```
绿鞋机制是港股 IPO 市场的一块"遮羞布"，也是一把"保护伞"。它能缓解阵痛，但治不了重病。
```

### ❌ 误区 6：图表没有说明

**错误示例**:
```
![古茗绿鞋消耗](https://raw.githubusercontent.com/zzpwestlife/writing/main/image/...)
```

**正确示例**:
```
![古茗绿鞋消耗](https://raw.githubusercontent.com/zzpwestlife/writing/main/image/港股IPO绿鞋机制实例解析/1769776675350.png)

```

## 主题风格对比

### 金融主题 vs AI 主题

| 维度 | 金融主题 | AI 主题 |
|------|----------|---------|
| **标题格式** | 双标题（主标题 + 副标题）+ emoji | 单标题（引导词 + 核心主题）+ emoji |
| **标题引导词** | 无 | "一文看懂"、"深度解析"、"全面了解" |
| **开头引入** | Blockquote + 系列导航 + 摘要 + 设问 | Blockquote（💡）+ 【导语】+ 比喻 |
| **章节划分** | 01/02/03 或 第一/二/三部分 | 阶段编号（第一阶段、第二阶段...）|
| **章节标记** | 数字编号 + emoji | Emoji + 阶段编号 + 阶段名称 |
| **特殊章节** | 使用 `---` 分隔 | 使用 【】标记（【导语】、【结语】、【互动话题】）|
| **内容驱动** | 案例驱动 + 数据支持 | 比喻驱动 + 列表说明 |
| **比喻使用** | 2-3 个生动比喻 | 贯穿全文的比喻（如"驾驶之旅"）|
| **视觉元素** | 图表 + 表格 + 公式 | 大量 emoji + 列表 + 配图建议 |
| **图表类型** | GitHub raw、Mermaid、在线图表 | 配图建议（描述性，非固定链接）|
| **互动元素** | 话题标签 + 免责声明 + 点赞引导 | 【互动话题】+ 投票选项 + 评论区邀请 |
| **结尾设计** | 总结 + 升华 + 系列预告 + 互动 | 【结语】+ 范式转移 + 展望未来 |
| **字数要求** | 1500-2500 字 | 1200-2000 字 |

### 主题选择指南

**使用金融主题风格，当**：
- 文章主题与金融、投资、IPO 相关
- 需要使用真实案例和数据支持
- 需要专业的风险提示和免责声明
- 内容偏严谨分析和论证

**使用 AI 主题风格，当**：
- 文章主题与 AI 技术、工具、人机协作相关
- 需要使用比喻和故事线讲解技术概念
- 需要设计互动话题和投票选项
- 内容偏技术科普和工具介绍

## AI 主题风格特点

### 标题风格

**格式**：
```
# [引导词] [核心主题] [emoji]
```

**引导词列表**：
- "一文看懂"、"深度解析"、"揭秘"
- "全面了解"、"彻底搞懂"、"全景复盘"

**Emoji 列表**：
- 🚗 进化、成长、旅程
- 🤖 AI、技术、工具
- 💡 启示、洞察、理解
- 🚀 未来、突破、创新

**示例**：
```
# 还在自己"开车"写代码？一文看懂人机协作的四个进化阶段
```
```
# 全面了解 ChatGPT：从原理到实战的完整指南 🚀
```

### 文章结构

**开头引入**：
```
> 💡 **摘要**：[一句话概括文章核心价值和亮点]

## 【导语】

[使用比喻引入主题，吸引读者注意]
```

**阶段划分**：
```
## 🚗 第一阶段：阶段名称

> **"[概括阶段特点的引用句]"**

**（配图建议：[描述性图片说明]）**

[阶段内容说明]

*   ✅ **优势**：[优势说明]
*   ❌ **痛点**：[痛点说明]
```

**结尾设计**：
```
## 【结语：范式的转移】

[总结文章核心观点]

在这场 [比喻] 中，我们最大的挑战不再是 [旧挑战]，而是：
1.  **[新挑战1]**
2.  **[新挑战2]**

[引发思考的问题或展望未来]
```

**互动元素**：
```
### 【互动话题】

**[互动问题]**

*   🚗 **选项一**：[选项说明]
*   🤖 **选项二**：[选项说明]
*   ✈️ **选项三**：[选项说明]
*   🚀 **选项四**：[选项说明]

*(欢迎在评论区分享你的体验！)*
```

### 内容特点

**比喻驱动**：
- 使用贯穿全文的比喻
- 比喻要与 AI 技术主题相关
- 使用生活化的例子解释技术概念

**列表说明**：
```
*   ✅ **优势**：[优势说明]
*   ❌ **痛点**：[痛点说明]
*   🗺️ **功能一**：[功能说明]
*   ⚠️ **提醒**：[提醒说明]
*   🧰 **功能二**：[功能说明]
```

**配图建议**：
- 每个阶段都有配图建议
- 配图建议是描述性的，非固定链接
- 帮助读者想象和配图

**格式**：
```
**（配图建议：[描述性图片说明]）**
```

### 视觉元素

**Emoji 装饰**：
- 大量使用 emoji 作为标记和装饰
- 每个 emoji 要与内容相关

**列表标记**：
- ✅ 优势
- ❌ 痛点
- 🗺️ 功能
- ⚠️ 提醒
- 🧰 功能

**常用 emoji**：

## 使用指南

### 如何使用 WeChat Writer Agent 创作 AI 主题文章

#### 方法 1：明确指定主题类型

```
请使用 wechat-writer agent 创作一篇关于 [主题] 的公众号文章，要求：
- 主题：ChatGPT 原理
- 主题类型：AI 技术科普
- 目标受众：普通读者
- 文章类型：技术科普
- 预计字数：1800 字
```

#### 方法 2：让 Agent 自动识别

```
请使用 wechat-writer agent 创作一篇关于 [主题] 的文章，agent 会自动识别主题类型。
```

### 主题识别规则

WeChat Writer Agent 会根据以下规则识别主题类型：

**识别为金融主题**，当：
- 关键词：IPO、投资、股票、港股、绿鞋、打新、基金、理财
- 目录结构：`contents/finance/`

**识别为 AI 主题**，当：
- 关键词：AI、ChatGPT、人工智能、机器学习、深度学习、GPT、LLM、编程、代码、工具、效率
- 目录结构：`contents/ai/`

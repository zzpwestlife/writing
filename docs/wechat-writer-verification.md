## 验证目标

验证 wechat-writer agent 是否能够：

1. **准确理解用户需求**：根据用户输入，生成符合要求的文章
2. **严格遵循写作风格**：按照写作风格指南创作文章
3. **生成高质量内容**：内容有价值、有深度、有专业性
4. **保持风格一致性**：与现有文章风格保持一致
5. **满足输出规范**：符合所有输出格式要求

---

## 验证方法

### 方法 1：对照写作风格指南验证

**步骤**：

1. **生成文章**：使用 wechat-writer agent 生成一篇文章
2. **逐项检查**：对照 `docs/writing-style-guide.md` 逐项检查
3. **打分评估**：根据检查结果打分
4. **记录问题**：记录不符合要求的地方

**检查清单**（来自 writing-style-guide.md）：

- [ ] 标题检查：双标题、emoji、吸引关键词
- [ ] 结构检查：开头、章节、结尾
- [ ] 内容检查：案例、数据、比喻
- [ ] 视觉检查：图表、排版
- [ ] 互动检查：标签、声明、互动

**示例代码**：

```python
# 验证脚本示例

def validate_article(article_path, style_guide_path):
    """验证文章是否符合写作风格"""

    # 读取文章和风格指南
    article = read_file(article_path)
    style_guide = read_file(style_guide_path)

    # 检查标题
    title_checks = validate_title(article)
    print(f"标题检查: {title_checks['score']}/100")

    # 检查结构
    structure_checks = validate_structure(article)
    print(f"结构检查: {structure_checks['score']}/100")

    # 检查内容
    content_checks = validate_content(article)
    print(f"内容检查: {content_checks['score']}/100")

    # 综合评分
    total_score = (title_checks['score'] + structure_checks['score'] + content_checks['score']) / 3

    return total_score
```

### 方法 2：与现有文章对比验证

**步骤**：

1. **选择参考文章**：从现有文章中选择一篇风格典型的文章
2. **生成对比文章**：使用 wechat-writer agent 生成相似主题的文章
3. **逐项对比**：对比两篇文章的风格、结构、内容
4. **评估一致性**：评估风格一致性

**对比维度**：

| 维度 | 对比项 | 一致性评分 (1-5) |
|------|--------|------------------|
| 标题风格 | 双标题、emoji、关键词 | ___/5 |
| 语言风格 | 口语化、比喻、通俗化 | ___/5 |
| 章节结构 | 编号方式、emoji 使用 | ___/5 |
| 内容组织 | 案例驱动、数据支持 | ___/5 |
| 视觉元素 | 图表、排版、公式 | ___/5 |
| 互动元素 | 标签、声明、引导 | ___/5 |

**评分标准**：
- 5 分：完全一致
- 4 分：基本一致，有细微差异

- 2 分：基本不一致
- 1 分：完全不一致

**通过标准**：平均分 ≥ 4.0

今天，我们就来拆解港股 IPO 配售规则的"黑箱"，看看新规到底改了什么，又该如何调整打新策略。

## 优化建议

### 常见问题及优化

#### 问题 1：标题不够吸引人

**表现**：
```
# 港股 IPO 配售规则新规解读
```

**优化建议**：
```
# 揭秘港股 IPO 新规：配售规则大调整，打新策略该如何应对？🧐
```

**说明**：添加"揭秘"、"大调整"等吸引关键词，添加emoji。

---

#### 问题 2：章节编号混乱

**表现**：
```
## 1. 什么是配售规则？
## 第二部分：新规解析
## Part 3：投资建议
```

**优化建议**：
```
## 01 什么是配售规则？
## 02 新规如何影响打新？
## 03 新规下的投资策略
```

**说明**：统一使用 "01/02/03" 或 "第一/二/三部分" 格式。

---

#### 问题 3：比喻不够生动

**表现**：
```
绿鞋机制是为了稳定股价。
```

**优化建议**：
```
绿鞋机制是港股 IPO 市场的一块"遮羞布"，也是一把"保护伞"。它能缓解阵痛，但治不了重病。
```

**说明**：使用"遮羞布"、"保护伞"、"缓解阵痛"、"治不了重病"等生动比喻。

---

#### 问题 4：数据支持不足

**表现**：
```
古茗的绿鞋消耗得很快。
```

**优化建议**：
```
古茗的绿鞋可谓是"鞠躬尽瘁"。数据显示，上市**第一天**，绿鞋资金就消耗了 **81.3%**！

数据来源：LiveReport 及市场公开数据
```

**说明**：添加具体数据和数据来源。

---

#### 问题 5：缺少风险提示

**表现**：
```
投资者可以积极参与打新。
```

**优化建议**：
```
⚠️ **风险提示**：
- 市场有风险，投资需谨慎
- 新股破发风险
- 资金流动性风险

投资者应理性评估风险，根据自身情况决定打新策略。
```

**说明**：添加明确的风险提示和免责声明。
